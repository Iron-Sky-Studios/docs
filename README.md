## Setup
### Python Dependencies

```bash
python -m virtualenv .venv

# Linux/Mac
source .venv/bin/activate

# Windows
.\.venv\Scripts\activate

pip install -r requirements.txt
```

### Pre-commit

```bash
pre-commit install
```

<br>
